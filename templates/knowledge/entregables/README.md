# Entregables

Todo lo que genera Claw (reportes, presentaciones, análisis, cursos) vive acá,
versionado en este mismo git. En fase 1 no se escribe nada de vuelta a Drive:
si algo tiene que subir, Daiana lo sube a mano (D-05).

Convención: `AAAA-MM-DD-<tema>/` o `AAAA-MM-DD-<tema>.md`, idioma según lo
pedido cada vez (D-19).
